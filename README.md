# ⚡ fastapi-alertengine

**Rugged, self-hosted observability and decision system for high-stakes FastAPI APIs.**

When a critical endpoint starts failing, dashboards don’t help.
You don’t need charts—you need **instant understanding and controlled response**.

AlertEngine was built for **[AnchorFlow / Tofamba](https://github.com/Tandem-Media/AnchorFlow)**, financial systems processing live mobile-money transactions where latency and failure directly translate into lost revenue.

It replaces SaaS-heavy observability stacks with a single principle:

**Detect → Understand → Decide → Act** (in seconds, not minutes).
